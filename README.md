### Inspiration

I really wanted to create something I could use for shortcuts while drawing in Clip Studio Paint!

### Challenges

Wiring the PCB to work and removing right angles was hard, but learning Fusion was harder.... Fusion oml...

### Specifications

BOM: 
- 6x Cherry MX Switches
- 1x XIAO RP2040
- 6x Blank DSA Keycaps
- 6x M3x16 Bolt
- 6x M3 Heatset
- 1x 0.91 inch OLED 

Others:
- KMK Firmware
- Top Case.stl 
- Bottom Case.stl
